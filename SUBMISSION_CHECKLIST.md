# Wiley Submission Checklist

Use this checklist before submitting to Materials and Processing (Wiley)

---

## Before You Begin

- [ ] Read the Author Guidelines thoroughly
- [ ] Ensure manuscript fits journal scope (materials science, mathematical frameworks, complex systems)
- [ ] Identify corresponding author with ORCID iD
- [ ] Gather all author information (names, affiliations, emails, ORCIDs)

---

## Required Files

### Main Submission
- [x] Manuscript PDF (`manuscript.pdf`)
- [x] Manuscript source file (`manuscript.tex`)
- [ ] Title page (if required by journal)
- [ ] Cover letter (`cover_letter.md` - customize before submission)
- [ ] Author declaration (`author_declaration.md` - complete before submission)

### Figures and Tables
- [ ] High-resolution figures (if separate from main text)
- [ ] Table files (if separate from main text)
- [ ] Figure captions included in main text

---

## Manuscript Content

### Structure
- [ ] Title is clear and descriptive
- [ ] Abstract is within word/character limit (check journal guidelines)
- [ ] Keywords included (5-7)
- [ ] Introduction provides context and motivation
- [ ] Methods/Theory section is complete
- [ ] Results/Analysis are thoroughly documented
- [ ] Discussion addresses implications
- [ ] Conclusions summarize contributions
- [ ] References are complete and in correct format

### Formatting
- [ ] Double-column format (if required)
- [ ] Page length within limits (8-10 pages for ETRI Journal)
- [ ] US spelling used (or consistent throughout)
- [ ] Mathematical notation clear and consistent
- [ ] Equations numbered consecutively
- [ ] Figures referenced in text (Fig. 1, not Figure 1 mid-sentence)
- [ ] Tables referenced in text

### Quality
- [ ] Language proofread and edited
- [ ] All abbreviations defined on first use
- [ ] Units in SI or SI-derived units
- [ ] Numbers 10+ use numerals
- [ ] Statistical measures identified (SD, SEM)
- [ ] Copyright permissions obtained for non-original material

---

## Required Statements

- [ ] Conflict of Interest statement (or declaration of no conflicts)
- [ ] Funding statement
- [ ] Data Availability statement
- [ ] Ethics statement (if applicable)
- [ ] CRediT author contributions
- [ ] ORCID iD for all authors (at least corresponding author)

---

## Submission System (ScholarOne)

1. [ ] Create account at https://mc.manuscriptcentral.com/madv
2. [ ] Log in to dashboard
3. [ ] Start new submission
4. [ ] Enter article details:
   - [ ] Article type (Research Article)
   - [ ] Title
   - [ ] Abstract
   - [ ] Keywords
   - [ ] Corresponding author info
5. [ ] Add all authors
6. [ ] Upload files in correct format
7. [ ] Assign files to correct categories (main text, figures, etc.)
8. [ ] Enter suggested reviewers
9. [ ] Enter excluded reviewers (if any)
10. [ ] Add comments to editor (if any)
11. [ ] Review and approve submission
12. [ ] Submit

---

## After Submission

- [ ] Save confirmation email
- [ ] Note manuscript ID number
- [ ] Check submission status periodically
- [ ] Respond promptly to any editor requests
- [ ] Prepare for reviewer comments

---

## Revision Round (If Requested)

- [ ] Address all reviewer comments
- [ ] Prepare point-by-point response letter
- [ ] Highlight changes in manuscript
- [ ] Upload revised files with clean versions
- [ ] Include response letter as separate file
